Gestão de Pacientes

A tabela de pacientes permite que você gerencie todos os atendidos pelo IBK. Abaixo estão as funcionalidades detalhadas e os botões responsáveis por ativá-las.

Adicionar Paciente

Utilize o botão principal da tela para abrir o formulário completo. Lembre-se de vincular um médico responsável.

Novo Paciente

Ver Detalhes

Ao clicar no ícone de "olho" na linha do paciente, você abre o modal contendo histórico clínico e triagens.

Ver Perfil

Inativação (Soft Delete)

Por exigências da LGPD, registros médicos não são apagados. Use o botão "Desativar" para ocultá-los.

Desativar

Controle de Profissionais

O gerenciamento dos profissionais que acessam o app AURA é vital para a segurança. Apenas Admins podem executar essas ações.

Vincular a Convênios

Para o médico aparecer no cadastro de um paciente, ele deve estar vinculado a um Convênio.

Gerenciar Vínculos

Gerar Senha de Acesso

Quando um médico esquecer a senha, vá ao perfil dele e gere uma nova credencial provisória de 6 dígitos.

Resetar Senha

Exportação e Relatórios

Todas as tabelas do sistema (Pacientes, Médicos, Logs) podem ser baixadas. Os filtros aplicados na tela afetam diretamente o resultado do arquivo final gerado.

Gerar Relatório

Clique no botão Exportar e aguarde a janela de pré-visualização para confirmar os dados.

Exportar

Formatos Suportados

Você pode baixar em PDF para relatórios limpos ou CSV caso precise abrir os dados no Excel.
